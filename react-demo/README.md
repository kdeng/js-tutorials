# Nest Image Tags Prototype

This repository is a prototype of the nest image tags.

## Getting Started

```sh
$ npm install
```

Start the local dev server:

```sh
$ npm run server
```

Navigate to **http://localhost:8080/** to view the app.

## Commands

### server

```sh
$ npm run server
```

```sh
$ npm test
```
